How It Works

  1. Choose a prompt below and copy it
  2. Paste into ChatGPT, Claude, or any AI assistant
  3. Have a conversation about movies you've watched
  4. Copy the CSV output when done
  5. Import to Letterboxd

After You Get Your CSV

  2. Save it as a .csv file (or paste directly)
  3. Go to letterboxd.com/import
  4. Upload your file and review the matches
  5. Confirm the import - done!
